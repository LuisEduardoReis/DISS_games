Introdução aos Jogos e à Programação em Javascript

Neste conjunto de tutoriais serão apresentados diversos jogos com o objetivo de dar ao leitor uma introdução aos temas de Programação e Desenvolvimento de Jogos.

Será usada a linguagem de scripting Javascript, uma das principais linguagens por detrás do funcionamento da web atualmente. Esta linguagem foi escolhida pela sua sintaxe genérica e facilidade de instalação e uso. Para complementar alguns dos exemplos será usado HTML para fazer uma interface simples. Para os exemplos gráficos será usada o p5.js, uma biblioteca que facilita o desenvolvimento de pequenos sketches.

Ao longo dos tutoriais serão apresentados brevemente conceitos que não se consideram muito importantes para uma introdução à programação e aos jogos, como por exemplo a linguagem HTML e diversos metodos para criar um pequeno servidor local. Mais detalhes sobre estes temas poderão ser encontrados em anexo.

  1. Rock Paper Scissors
  2. Hangman
  3. Snake
